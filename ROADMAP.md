## 향후 개발 계획

### 1. 음성 구간 클릭 재생 기능
- **설명**: 타임라인 그래프에서 음성 구간을 클릭하면 해당 부분을 재생
- **필요 사항**:
  - 오디오 재생 라이브러리 (PyQt6 QMediaPlayer 또는 pygame)
  - 타임라인 위젯에 클릭 이벤트 처리 추가
  - 세그먼트별 시작/끝 시간 정보 필요 (현재 보유 중)

### 2. 세그먼트 리포트 내보내기
- **설명**: 음성/비음성 구간을 텍스트 파일로 내보내기
- **포맷 예시**:
  ```
  00:00:00.00-00:00:21.34 (Speech)
  00:00:21.35-00:00:29.81 (Non-Speech)
  ```
- **필요 사항**:
  - 내보내기 버튼 또는 메뉴 추가
  - 파일 저장 다이얼로그
  - 세그먼트 데이터 포맷팅 로직

## 기술적 고려사항

### 긴 오디오 파일 시각화 문제 (해결 필요)

**문제**: 226분 오디오에서 타임라인이 거의 100% 파란색으로 표시됨 (실제 58.5%)

**원인**: 세그먼트별 최소 너비(1px) 확장으로 인해 짧은 silence 구간이 덮어씌워짐

**해결 방안 (향후 구현)**:

| 방안 | 설명 | 장점 | 단점 |
|-----|------|-----|-----|
| **줌/스크롤 기능** | 타임라인을 확대/이동 가능하게 | 세그먼트 병합 불필요, 정밀 탐색 가능 | UI 복잡도 증가 |
| **시각화 전용 병합** | 렌더링 시에만 인접 세그먼트 병합 | 원본 데이터 보존, 클릭 재생 가능 | 구현 복잡도 |
| **미니맵 + 상세뷰** | 전체 개요 + 선택 구간 상세 표시 | 직관적 탐색 | 화면 공간 필요 |

**권장 방향**: 줌/스크롤 기능 + 원본 세그먼트 데이터 보존
- 타임라인 클릭 시 세그먼트 데이터로 정확한 위치 계산 가능
- 줌 인 시 세그먼트 경계가 명확히 보임
- 클릭 재생 기능과 호환
